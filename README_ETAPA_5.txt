ETAPA 5 - Projeto preparado para GitHub

O que foi feito:

1. README.md organizado para o GitHub.
2. .gitignore revisado.
3. Instruções para criar .env local.
4. Instruções para criar public/js/config.js local.
5. Passo a passo para iniciar o Git e subir o projeto.

IMPORTANTE:

Não envie estes arquivos para o GitHub:

- .env
- public/js/config.js
- node_modules/

Antes de fazer commit, rode:

git status

Se .env ou public/js/config.js aparecerem na lista, pare e corrija antes de subir.
